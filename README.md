# Backend – Sistema SELLOS-G

Este repositorio contiene el **backend del sistema SELLOS-G**, desarrollado con **Node.js**, **Express** y **MongoDB**.  
El backend gestiona la lógica del negocio, usuarios y roles, productos, pedidos, clientes y autenticación, y permite la comunicación con el frontend y herramientas de prueba.

Este documento describe los pasos necesarios para **instalar, configurar y ejecutar** el backend del sistema.

---

## Tecnologías utilizadas

- Node.js  
- Express  
- MongoDB  
- JSON Web Tokens (JWT)  
- Nodemailer  

---

## Requisitos

Antes de iniciar, el equipo debe contar con lo siguiente:

- Node.js **v18 o superior**
- npm (gestor de paquetes de Node.js)
- MongoDB Community Server o acceso a **MongoDB Atlas**
- Visual Studio Code (editor recomendado)
- Postman para pruebas de endpoints
